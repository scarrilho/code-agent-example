## About

This is a simple coding agent built with **Koog**.

It is inspired by the article  
[How to Build an Agent](https://ampcode.com/how-to-build-an-agent).

The Koog setup is based on the examples from  
[koog-tutorials](https://github.com/svtk/koog-tutorials).
